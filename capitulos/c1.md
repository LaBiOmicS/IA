## 1.4 O que √© Aprendizado de M√°quina?

O **Machine Learning (ML)** √© um subcampo da IA que permite que computadores **aprendam a partir dos dados**.

### Programa√ß√£o tradicional vs ML

```python
# ----------------------------
# EXEMPLO 1: Programa√ß√£o tradicional
# ----------------------------

    # Diagnostico simples baseado em uma regra fixa:
    # Se a temperatura > 38¬∞C, o paciente tem febre.
    # Caso contrario, esta normal.
    if temperatura > 38:

    else:
```

No modelo tradicional, **n√≥s definimos a regra**.  
No ML, deixamos o algoritmo **aprender com exemplos**:

```python

# ----------------------------

from sklearn.linear_model import LogisticRegression

# Fazendo previsao

print(modelo.predict(nova_temperatura))  # saida: [1] -> provavel febre
```

## 1.8 Fluxo de Trabalho em ML

1. **Coletar dados** ‚Äì bases gen√¥micas, imagens m√©dicas, prontu√°rios.  
2. **Preparar dados** ‚Äì limpar, normalizar, transformar.  
3. **Explorar (EDA)** ‚Äì entender padr√µes.  
4. **Treinar** ‚Äì ajustar o modelo.  
5. **Avaliar** ‚Äì medir desempenho.  
6. **Ajustar** ‚Äì refinar par√¢metros.  
7. **Implantar** ‚Äì usar em aplica√ß√µes reais.

üìå **Exemplo com pipeline simplificado:**

```python

from sklearn.metrics import classification_report

# Dataset ficticio

y = [0, 1, 1, 0]

# Divisao em treino e teste
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.5, random_state=42)

# Modelo
modelo = DecisionTreeClassifier().fit(X_train, y_train)

# Avaliacao
y_pred = modelo.predict(X_test)
print(classification_report(y_test, y_pred))
```
